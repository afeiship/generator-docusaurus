# next-finxos-element-type
> Element type/types.

## installation
```bash
npm install -S afeiship/next-finxos-element-type --registry=https://registry.npm.taobao.org
```

## apis
| api  | params | description                   |
| ---- | ------ | ----------------------------- |
| get  | -      | Get by fileld(default: value) |
| gets | -      | Get all types                 |
